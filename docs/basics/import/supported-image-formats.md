# Supported Image formats

Below are the supported file formats for Hyperspectral and RGB images.

| Hyperspectral images     | RGB images     |
| ------------------------ | -------------- |
| ENVI .dat/.raw with .hdr | .png           |
| .tif or .tiff            | .jpg or .jpeg  |
|                          | .tif  or .tiff |
|                          | .bmp           |
